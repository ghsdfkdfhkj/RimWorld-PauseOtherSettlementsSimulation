New Features

Local Time System (Real-Sim):
- **True Local Time:** Implemented a robust local time system. When a settlement is paused/simulated, its time (ticks) effectively halts relative to the world.
- **Environment Sync:** Sun position, light levels, and seasonal temperatures now correctly reflect the local time of the settlement when you enter the map. No more "time jumps" to global time upon loading.
- **Seamless Transition:** Logic added to handle map switching, ensuring pause durations are accurately tracked even when moving between multiple settlements.

UI Indicators:
- **Local Time Display:** Added a "Local Time" column to the mod's main tab (Simulation Control). You can now see the exact date/time for each settlement.
- **Time Offset:** Added an "Offset" column showing how far behind the global time a settlement is (e.g., "-5.2d").
- **Off-World Support:** The UI now correctly detects and displays information for special maps like **SOS2 Ships**, Space Stations, and generic Camps, which previously had missing or incorrect data.

Bug Fixes

- **Initialization Crash:** Fixed a critical "No active language!" error that occurred during startup by correcting the initialization order of Harmony patches.
- **Time Calculation:** Fixed an issue where local time calculation would drift or reset when switching maps.
- **Data Persistence:** Fixed save/load issues ensuring pause states and time offsets are preserved across game sessions.
- **NullReferenceException:** Fixed a crash on game load caused by uninitialized world component data.

새로운 기능

로컬 타임 시스템 (리얼 시뮬레이션):
- **진정한 시간 정지:** 정착지가 시뮬레이션(일시정지) 상태일 때, 해당 정착지의 시간이 실제로 멈추도록 구현했습니다.
- **환경 동기화:** 맵에 진입할 때 태양 위치, 밝기, 계절별 기온이 글로벌 시간이 아닌 정착지의 "로컬 시간"에 맞춰 적용됩니다. 더 이상 로딩 시 강제로 시간이 흐르는 현상이 발생하지 않습니다.
- **자연스러운 전환:** 맵 이동 시 정지된 시간이 정확히 계산되도록 로직을 보완했습니다.

UI 정보 표시:
- **로컬 시간(Local Time):** 모드 메인 탭에 각 정착지의 현재 날짜와 시간을 표시하는 열을 추가했습니다.
- **오프셋(Offset):** 글로벌 시간 대비 얼마나 시간이 지났는지(예: -5.2일)를 보여주는 오프셋 정보를 추가했습니다.
- **우주/특수 맵 지원:** **SOS2 함선**이나 우주 정거장, 임시 캠프 등 타일 정보가 특수한 맵들도 이제 정상적으로 인식되어 목록에 표시됩니다.

버그 수정

- **초기화 충돌 해결:** 게임 실행 시 발생하던 "No active language!" 오류를 수정했습니다. 하모니 패치 순서를 조정하여 안정성을 확보했습니다.
- **시간 계산 오류:** 맵을 오갈 때 시간이 어긋나거나 초기화되던 문제를 해결했습니다.
- **데이터 저장:** 게임 저장/불러오기 시 정착지의 시간 정보가 올바르게 유지되도록 수정했습니다.
- **NullReferenceException:** 게임 로딩 시 데이터 초기화 문제로 인한 크래시를 수정했습니다.
